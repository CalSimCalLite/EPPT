Project Operations

Create a new project:

Click on the File | Project | New or type the 'Ctrl' and 'n' keys together. If a dialog box appears as a result, click 'Yes' to save the currently open project before it closes, or 'No' to close it without saving.

Alternatively, if a new project is to created from altering an existing project, Click on the File | Project | Open or type the 'Ctrl' and 'o' keys together instead. Use the 'Open' file selector a file dialog box will pop up, allow you to find and select open the desired project. The new project must be saved with a different name than the existing project.

Open an existing project:

See the second paragraph above.

Save a project:

Click File | Project | Save/Save As to save the open project. Choose Save to save the project with its current name and location (i.e., overwrite the last saved version of the open project); choose Save As, to save the project with a different name and/or location. Choosing Save As will result in a 'Save' file selector box. Use this box to choose the name and location at which to save the project.

Merge with an existing project:

If a project is not currently open, Click on File | Project | Open (or New) to open an existing project or create a new one.

Click on the File | Project | Merge With.... In the Open file selector, find and select a file and merge it with the open project.

If any portion of the open project is different than that portion of the project being merged (e.g., the Base DVAR files are different), the open project's portion will be used.
