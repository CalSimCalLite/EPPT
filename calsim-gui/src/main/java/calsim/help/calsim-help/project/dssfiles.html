Opening Decision Variables and State Variables Files

Click on the Output tab if it's not already selected. Click the 'DV' command button on the Base Files line. Use the Open file selector window to find and select the base decision variables (DV) file. Click the 'SV' command button on the Base Files line. Use the Open file selector window to select the base state variables (SV) file (it's in the same directory as the selected DV file). If you want compare the results of two to four model runs, then click on the 'DV' and 'SV' command buttons and load the files on the appropriate number of Comp Files lines. Alternatively, The 'File' menu can be used. Click File | Open Base, Open Comp1, Open Comp 2, and/or Open Comp 3 to select and load the desired DV and SV files.

Note: After opening an existing project you do not need to open DV and/or SV files unless you want to alter the open, existing project or create a new project which has some DV and/or SV files in common with the opened project. The DV and SV files' names and paths associated with an open project are present in the text boxes next to the DV and SV buttons. If you want to alter the open project or create a new project, select DV and SV files on the lines you want the change and follow the procedure in the paragraph above.
