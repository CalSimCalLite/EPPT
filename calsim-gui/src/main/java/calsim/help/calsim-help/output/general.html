General Retrieval of Timeseries Data

Any time series data in a DSS file may be retrieved through the Output | General tab. The data type retrieved is either decision variables (DVAR) or state variables (SVAR). Chose DVAR in the 'TYPE' box to view output. Pressing the "Filter" button with nothing in A through F boxes to list all pathnames available in this file. Select (highlight) the variable line desired, click the desired mode of output viewing, then click the 'Retrieve' button to retrieve and view the data.
