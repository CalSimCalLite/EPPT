How to Use a DTS Tree

The DTS and MTS definitions are that are stored in the .tree file are placed in a list when the tree file or the project are opened. A DTS or MTS is displayed in the DTS Table when a terminal node is selected. A Terminal node is a node that ends with .dts or .mts and does not allow nodes to be added to it. A Folder is a node that can have other terminal nodes or other folders added to it.

By right clicking on the DTS Tree window a popup menu will appear and the node closest to where the window was clicked will automatically be selected.

DTS Tree Window

The menu options perform the following actions:

Open - Displays the time series of the selected DTS or MTS in the view checked in the message panel.

Add DTS - Adds a DTS node to the tree and creates a blank DTS in the DTS Table.

Add MTS - Adds a MTS node to the tree and creates a blank MTS in the DTS Table.

Add Folder - Adds a Folder node to the tree.

Imports DTS - Adds a DTS node to the tree and that has been saved as a file.

Imports MTS - Adds a MTS node to the tree and that has been saved as a file.

Save - Saves the selected DTS or MTS as a file.

Rename - Starts the editing session to change the name of the node and DTS.

Delete - Removes the selected node from the tree and from the DTS list.

Cut - Removes the node from tree and allows it to be pasted in another directory.

Copy - Makes a duplicate of the selected node and the duplicate can be pasted into another directory.

Paste - Paste inserts a new node in the selected folder from the node that was cut or copied.
