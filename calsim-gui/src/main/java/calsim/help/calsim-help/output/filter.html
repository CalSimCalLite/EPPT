Filtering the DSS Data

Select General tab in the central panel of the Output tab. Type in the specific pathname part(s) in the corresponding text field(s) to filter the DV or SV variables loaded in the table. Use * (a wildcard character) to represent any number of characters.

For example, choosing DVAR for the type of variables and typing S* in the B-Part box (labeled 'B') and clicking on the Filter button will display all the path names with B-Part starting with 'S'. After doing this, the C-Part for the variables in the table will be Storage, Storage Zone, etc.

For another example, choosing DVAR and typing 'Storage *' in the C-Part box (labeled 'C') and clicking on the Filter button will display all 'Storage' variables only. Variables with C-Part of 'Storage-Zone' will not be displayed in the table due to the blank space before the *.
