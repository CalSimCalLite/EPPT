Goal Statement: Short Form

There is a even more concise form for expressing traditional LP constraints. Omit all tags except the goal tag and specify a comparison expression using <, >, or =. This is the preferred method for specifying constraints without associated conditions or penalties.

 
goal tagname {
     expr1 < | = | > expr2
}
Example: 
goal tracyLessThanAllow { D418 < TracyMaxAllow}
