Prepare a project for study

To prepare a project for study, you must follow the following steps:

1. Click on File | Project | New (or Open) to open a new or existing project. *Please refer to "About a project" tutorial.

2. Click in File | Open Base to open base file(s), include SVAR (input) file and DVAR (output) file. If you want compare two model run, then you need also open compare files. *Please refer to the "Loading a model run" tutorial.

Note: If the dss files associated with this project were opened at the last time when you worked on this project, and you saved this project appropriately, then you don't need to open them again. Anyway, if you need to open a new dss file, you may do so now.

3. (Optional) You could load an image associated with the current project. Click on the File | Load Image, a file dialog box will pop up, choose an image file to open, the image will be displayed in the center of the frame.

4. Click on File | Open Connectivity to load a connectivity file of the project, for the purpose of computing mass balance.

5. Click on the Edit menu, to set time window and units. Select a units and click on the Time Window to edit the time window. *Please refer to the "Setting a time window" tutorial.

6. Click on the View menu to choose a data view type. *Please refer to the "Changing display settings".

7. Click on the Mode menu to select a mode. *Please refer to "Setting the mode" tutorial.

Now, you are ready to analysis this project by using the rest of the menus.
