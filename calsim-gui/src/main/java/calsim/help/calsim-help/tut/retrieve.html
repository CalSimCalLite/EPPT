Retrieving data by node:
Select the Node/Arc/General toggle button first, then click on the Node Menu. The first item in the menu would be a text field initialized to some node number. Edit this number to the node number for which you want the output displayed. Then select the appropriate item from the other menu items under the Node Menu list.

Retrieving data by arc:
Select the Node/Arc/General toggle button first, then click on the Arc Menu. The first item in this menu is a text field initialized to the previous selection. Enter the Arc number for which the output is desired and then click the appropriate item from the other menu items in the Arc Menu list.

 
Note: If you followed this tutorial step by step, then your project should be ready to do the following example now, otherwise, you need set your project ready first. Please also see "Prepare a project for study".

Example1 Click on Node, change the node number to 1, click on the Storage EOP, you should see the storage data of node 1 is plotted.

Example2 Click on Arc, change the node number to 9, click on the Channel Flow, you should see the data of channel arc C9 is plotted.
