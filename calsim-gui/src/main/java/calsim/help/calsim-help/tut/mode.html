Setting the mode:

This menu provides the choices for the displayed output. The base mode determines that the output of base study is displayed. The compare mode determines that the outputs of both base study and comparison study are displayed. The difference mode determines that the difference between bast study and comparison study is displayed.

    Click on the Edit menu, then click on Mode|Base or Mode|Compare or Mode|Difference for the output. For now just leave the Model|Base selected as we will do comparative analysis later in the tutorial.
