Edit a schematic

Click on File | Edit Schematic, a file dialog box will pop up, open the schematic file that you want to edit, now the schematic should be displayed in a schematic editor. All the editing tools are at the bottom of the editor. The first icon let you move nodes and its associated arcs, the rest icons let you create new objects and require a node/arc id.

 

Note: You should always set the project to ready first. Please see "Prepare a project for study".
